前言

this 離開物件就沒有意義，因為 this 指向的是它本身目前所處的一個 空間，
而這個空間等同於容納 this 的這一個 物件。

由於 JS 是物件導向設計的語法，透過以下最簡單的例子，

函式的原始參數

除此之外，也可透過函式原先的幾個參數來觀察：

  • params (呼叫函式第一個帶入的參數，1)
  • this 指向 window
  • window
  • arguments 預設是帶入所有參數，變成一個類陣列 [1, 2, 3]

this 指向物件

this 指向的是不是 window，
端看調用它的方法，跟 this 所處的作用域無關，
因此 this 可視為存在於「動態」的作用域 (?)
在以下的調用方法中，this 都會指向物件。


this 指向全域
