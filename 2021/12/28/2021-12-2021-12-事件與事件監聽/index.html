什麼是事件 (Event)

事件 是觸發一段 JS 函式的方法，
也可以說，事件是用戶和瀏覽器互動的瞬間，
互動有很多種，像是鍵盤事件、滑鼠事件 (click / dblclick / mouseenter)…，
基本上如同滑鼠的 hover 之於觸發 CSS 的效果。


何謂事件監聽及其兩種寫法

事件監聽器 (addEventListener) 主要用來偵測使用者在網頁頁面上的所有行為，
舉凡：點擊按鈕、滑鼠滑過的區域、視窗放大縮小等，都可被偵測。


寫在 HTML 標籤屬性上 (不建議做法)

在按鈕標籤上，增加一個 onclick 屬性，並寫下對應的函式名稱，
這樣就寫完了一個監聽事件，點擊按鈕時就會觸發 sendPassword() 函式。

註 在標籤上的 this，指的就是 button 這個元素本身。
注意！ 為避免 XSS 攻擊事件，script 標籤別放在 HTML 上面，網站很容易被攻擊！


透過 JS 綁定 DOM

透過 JS 來綁定，將 DOM 儲存在變數裡，對 DOM 綁定監聽事件。


拆解監聽器結構

event (e)

在瀏覽器中，有事件被觸發時，event 會變成函式的參數，
原因是這個事件本身也是一個物件，內含瀏覽器相關的資訊，
例如：滑鼠座標、滑鼠滾輪上下滾動的距離、鍵盤的按鍵…等，
帶入函式後就能利用這事件內建的資訊了。


target 監聽對象

即瀏覽器，若要選取範圍可透過 class、ID。


type 事件類型

通常是一個觸發事件，例如滑鼠或鍵盤事件。


function

要透過滑鼠或鍵盤事件去執行的動作。
