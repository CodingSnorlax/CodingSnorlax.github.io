sort 陣列使用時機

sort() 可以接受帶入 函式參數，
透過這個函式參數來設定 sort() 的排序。


字串排序

預設是根據字串的 Unicode 編碼位置而定。


數字排序

假設有一個陣列，裡面有五個數字，
希望透過陣列整理出 大到小 排序，
透過函式，任選 a, b 兩個數字相減：

  • 若 a - b > 0 則 a 的順序在 b 之前
  • 若 a - b < 0 則 b 的順序在 a 之前
  • 若 a - b = 0 則 a, b 不動
