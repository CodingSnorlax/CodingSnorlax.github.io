setAttribute 增設屬性與值

setAttribute 是可以同時用來增設 HTML 元素上的屬性以及屬性的值，
由於這樣的特性，setAttribute 可以用來做很多事。


setAttribute 的寫法


setAttribute 增加 CSS 樣式

搭配 CSS，就可以透過 setAttribute 來更改 HTML 元素的樣式！

底下是一個 title text 由一般的 div 包裹著，它的文字要當成 title 實在太小了，
如果不更動 HTML，只用加一個 class 的方式來改變字級，應該怎麼做？

透過 setAttribute 增加 element 的屬性與值，
title text 的字級就會很明顯的變大了！


setAttribute 在 <a> 增加網址

  • 用 setAttribute 在綁定 DOM 的 <a> 上，修改它的 href 值，增加 google 網址
  • 用 setAttribute 在 <a> 上增加 target 屬性，增加 ‘_blank’ (跳轉網頁)
