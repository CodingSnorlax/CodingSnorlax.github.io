在寫 JavaScript 的時候，要綁定 DOM 一定會頻繁用到 querySelector 和 querySelectorAll，
因此這篇要來紀錄一下這兩者的用法，以下也搭配 DOM 結構的示意圖。




querySelector (選取 1 個 / 第 1 個 DOM)

只會選取一個 id 或 class，可用來綁定單一的 id 或 class，

querySelectorAll (選取多個標籤組成 nodelist)

如它的名稱所示，既然是 querySelectorAll()，
根據傳入的 class 或屬性名稱，它就可以用來一次同時綁定多個同名 class 或屬性，

小結

  • querySelector：選取單一個元素。
  • querySelectorAll：選取多個元素。

備註 同時還可以利用 querySelectorAll 組成 nodelist 的特性，來串接或修改資料，是很好用的一個功能。
備註 nodelist 遇到有一些陣列方法不能使用，可以先把它轉成陣列，方法可參考 NodeList 與 Array 差異
