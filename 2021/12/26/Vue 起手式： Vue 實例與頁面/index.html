Vue 起手式–實例 (Instance)

在 Vue 文件內的 實例 指的就是 物件，
data 即是所有動態資料存放的位置，而其屬性名稱是對應到 HTML 的插值。
接著 Vue 實例內的資料，需要透過綁定到 DOM 上，才能渲染到畫面，
以下個別拆解出 element 及 data 寫法的意思。

Vue instance 綁定 DOM

Vue 實例綁定 DOM 的語法是： Vue.createApp({}).mount() 。
基本上 Vue 語法的 .mount('#app')，
就是在說要把 Vue 實例安裝到 #app 這個 DOM 上，
是不是很像原生 JS 的 document.getElementById(‘app’)。

Vue 實例的 data 物件

Vue 3 的 data 物件包在函式裡，而這樣的寫法，是為日後的組件作準備。

如果我們在 data 函式內，看看 this 指向哪，
會發現這個 this 是指向 Vue 實例 (Vue 在調用這個 data 函式)，
因此，此處 data 絕對不可以寫成箭頭函式。

備註 所有被 Vue instance 託管的函式，都不可寫成箭頭函式，this 會變成指向 window。


Vue 起手式–頁面

完成設置 Vue 實例後，還需要在 HTML 上設定一個容器承接資料：

  • 一個 Vue instance 只能對應一個容器。
  • id 選取器上面的 app 就是綁定在 .mount()。

插值語法 (Interpolation)

主要用於控制 HTML 開始標記 到 結束標記 之間的 文字。

  • {{}} 稱之為插值，是頁面上接收 data 屬性的位置。
  • 僅接受 表達式，以下當 Vue instance 的 greetings 資料為 false (不存在) 時，會顯示 Welcome!。

指令語法

  • 主要用於控制 HTML 上的 標籤，範圍包括：標籤屬性、標籤的文字內容、綁定事件。
  • 指令語法以 v- 開頭，v-bind、v-model…。
  • 指令語法會使得範例的：linkTitle、link 轉為一個 表達式 (因為它們在 Data 內都有對應的值)。
