MVVM 模型圖


Model (資料)

資料獨立存放處，可在此處動態更新資料，
model 上的資料，全部都會被加入到 View Model，
因此資料都可同步透過 View Model 解析到 View，
這概念就是來自 關注點分離，便於管理資料。


View (視圖、模板)

在 {{}} 可接收到 View Model 這整大包物件上的所有資料，
除此之外，插值的位置只接受表達式才能在頁面上產生更動。


View Model (視圖模型)

也就是 Vue 實例，類似於通道、隧道的概念，串起 model 跟 view 間的通道，
只要更動到 Model 內的資料，必定要通過 View Model 這條路徑。

  • Data Bindings: 將資料綁定到頁面上指定的位置，例如 v-bind 單向綁定。
  • DOM Listeners: 對 DOM 事件監聽，例如 v-model 雙向綁定。

備註 在 Vue 裡面，vm 縮寫指的就是 view model。
