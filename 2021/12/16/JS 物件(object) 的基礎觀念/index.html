在開始看物件取值之前，再看一看物件的基本定義吧！

物件的結構

屬性(key) ：用來取得資料的「位置」。
值(value) ：有不同型別。

在物件內，沒有順序性，只有屬性名稱與值。


物件的功能

物件是 JavaScript 的物件型別之一，理解它的方法就跟在認識真實世界中的實體物件一樣，
JS 的物件內會有很多屬性，而屬性可以是一個原始型別的值、物件或陣列，這些資料可以構成一項物品的詳細資訊。

重點中的重點：物件主要在記述詳細資訊
