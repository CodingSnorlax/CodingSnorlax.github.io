什麼是 AJAX?

全名為 Asynchronous JavaScript and XML，最核心的語法部分是 XMLHttpRequest。

在頁面上即便沒有重新整理網頁，透過 AJAX 也能用瀏覽器撈取遠端資料庫，
並即時同步回傳後端資料庫的資料，使得網頁能即時更動、回應。


取得 AJAX 資料的方法

發展到現今已有三種方法都可發請求，取得 AJAX 資料：

  • XMLHttpRequest
  • fetch
  • axios (操作底層原理是 XMLHttpRequest)

AJAX 非同步特性

非同步特性不會等到資料回傳，而是在資料尚未回傳前，
持續讓程式碼繼續往下跑完，對使用者來說，等待資料的過程，
畫面上還是有其他資訊可以觀看或有介面可以操作，
等伺服器回應後，就得到回傳的資料。


AJAX 可發出的請求格式

  • get (讀取)
  • post (傳送資料到伺服器)
  • patch (修改)
  • put (修改)
  • delete (刪除)

XMLHttpRequest 原生語法

透過以下第 2 行程式碼的宣告，產生一包新的 XMLHttpRequest 物件，
能開始透過瀏覽器和其他伺服器要資料。


readyState 請求狀態

透過 readyState 查詢目前狀態 (再 call 一次 xhr 可觀察到 readyState)：

  • readyState: 0 => 已成功建立 XMLHttpRequest 物件，但尚未撈取資料。
  • readyState: 1 => 用了 xhr.open()，但尚未送出。
  • readyState: 2 => 已發出 xhr.send()。
  • readyState: 3 => loading
  • readyState: 4 => 已成功撈取資料。

onload：待資料回傳後啟動下一步

onload 是等到資料回傳後(也就是 readyState: 4)，
才會觸發去執行 function 的關鍵語法。


XMLHttpRequest 發請求流程整理

  1. 建立 XMLHttpRequest
  2. 傳送請求到對方伺服器要資料
  3. 資料回傳到自己的瀏覽器
