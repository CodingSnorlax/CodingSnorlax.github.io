map 陣列使用時機

  • map 與 forEach 很相似，都會遍歷陣列內的每一筆資料。
  • map 會 return 一筆新的陣列資料，適用於對陣列資料逐筆做修改時。

備註 forEach 不會 return 一筆新的陣列資料。
